A variable in JS is anything that can be stored and changed later on. In JS we declare variables with these keywords Var,let,const

In JS we must have identifiers for variables which are givem unique names. These unique names can contain:

  • letters,digits, and underscores
  • names are case sensitive (x and X are different variables)
  • names can begin with a letter

In JS we have an assignment operator for example the equal sign "=" which is not an "equal to " operator.However in javascript "equal to" can be wrtitten as "=="

For example x = x + 5 does not make sense algebraicly however in JS it makes sesne because it assigns the value of x = x + 5. This is because it calculates the value of the statement and puts the results into x and the value of x incremenrts by 5.
