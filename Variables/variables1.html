Variables:

A variable in JS is anything that can be stored and changed later on. In JS we declare variables with these keywords Var,let,const

We will use Var

  • Create a variable called x, assign the value 5 to the Varible. In this case x stores the value of 5

  • Create a variable called y, assign the value 6 to the Varible. In this case y stores the value of 6

  • Create a variable called z, assign the value x and y and add them togther . In this case z stored the combined value of x and y which is 11
